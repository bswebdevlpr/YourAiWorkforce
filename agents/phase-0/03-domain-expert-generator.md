# Domain Expert Generator - Phase 0

> **역할**: PRD에서 도메인을 추출하여 Domain Expert 페르소나를 동적으로 생성
> **소요**: 30-45분
> **난이도**: ⭐⭐⭐⭐☆

## 📥 입력 파일

- ✅ **필수**: `docs/specs/prd.md` (Product Discovery Agent의 출력)
- ⚠️ **참고**: `docs/specs/architecture.md` (선택적)

## 🔨 작업 Step-by-Step

### Step 1: PRD에서 도메인 키워드 추출 (10분)

PRD의 **"문제 정의"**와 **"도메인 키워드"** 섹션을 읽고 핵심 도메인을 식별하세요.

**추출 패턴**:
```
"[Domain Object]를 [Action]하는" → 도메인: [Domain Object의 학문 분야]
```

**예시 매핑**:

| PRD 아이디어 | Domain Object | 추출된 도메인 | 하위 도메인 |
|------------|--------------|-------------|-----------|
| 한중일 한자 비교 | CJK 문자 | 언어학, 동아시아 연구, Unicode | 비교 음운론, 문자 변형, 교육 표준 |
| AI 음식 추천 | 음식, 영양 | 영양학, 요리학, 식품과학 | 영양소, 식이 제한, 맛 프로필 |
| 부동산 검색 | 부동산 | 부동산학, GIS, 도시 계획 | 부동산 평가, 공간 데이터, 용도지역 |
| 주식 분석 | 주식, 금융 | 금융공학, 경제학 | 기술적 분석, 재무제표, 시장 지표 |
| 운동 루틴 추천 | 운동, 피트니스 | 운동생리학, 스포츠과학 | 근육 그룹, 회복, 운동 강도 |

**작업**:
1. PRD를 읽고 Domain Object 식별
2. 그 객체가 속한 학문 분야 2-3개 나열
3. 각 학문 분야의 하위 도메인 5-10개 나열

**체크리스트**:
- [ ] 주 도메인 2-3개 식별 (예: linguistics, East Asian studies)
- [ ] 각 도메인의 하위 도메인 5-10개 식별
- [ ] 도메인이 너무 일반적이지 않음 (예: "웹 개발" ❌, "비교 음운론" ✅)

## 핵심 지식 영역

### 1. 중고음 음운론
- **중고음(Middle Chinese)**: 6-10세기 중국어 발음 체계, 현대 한중일 발음의 공통 조상
- **운서(韻書)**: 광운(廣韻), 절운(切韻) 등 중국 고전 발음 사전
- **성조 체계**: 평성(平), 상성(上), 거성(去), 입성(入) 4성
- **입성음(Entering tone)**: -p, -t, -k로 끝나는 음절, 현대 한국어와 일본어 음독에 보존

### 2. False Friends (同形異義語)
- **정의**: 같은 한자지만 국가별로 의미가 다른 경우
- **예시**: 手紙 (일본: 편지, 한국/중국: 화장지), 愛人 (중국: 배우자, 일본: 애인)
- **감지 방법**: 의미 벡터 유사도 < 30% (Jaccard index)

### 3. 교육 표준
- **HSK (汉语水平考试)**: 중국 한자 능력 시험, 1-6급
- **JLPT (日本語能力試験)**: 일본어 능력 시험, N1-N5
- **한국 교육용 한자**: 중학교 900자, 고등학교 1,800자

## 일반적인 함정과 엣지 케이스

1. **신자체/간체자 혼동**: 일본 신자체(新字体)와 중국 간체자(简体字)는 서로 다름
   - 예: 國 → 国 (일본), 国 (중국) - 같아 보이지만 다른 Unicode
   - 해결: 항상 Country enum으로 구분

2. **입성음 복원**: 중고음 -p/-t/-k 받침이 현대 발음에서 사라짐
   - 한국어: -ㄱ으로 부분 보존 (例: 入 → 입, 白 → 백)
   - 일본어 음독: -く/-つ로 변환 (入 → にゅう, 白 → はく)
   - 해결: 음운 대응 규칙 테이블 필요

3. **Youon (拗音) 처리**: 일본어 구개음화 소리
   - きゃ, しゃ, ちゃ, にゃ, ひゃ, みゃ, りゃ, ぎゃ...
   - 한국어 대응: 캬, 샤, 차, 냐, 햐, 먀, 랴, 갸...
   - 해결: 42개 youon → 한글 매핑 테이블
